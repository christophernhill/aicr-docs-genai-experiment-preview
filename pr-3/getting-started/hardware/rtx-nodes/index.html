    GPU Getting Started RTX PRO 6000

    NVIDIA RTX PRO 6000 GPU Nodes¶

    The RTX nodes provide broad-access GPU computing — 19 Dell PowerEdge XE7745 air-cooled servers, each containing 8 NVIDIA RTX PRO 6000 Blackwell Server Edition GPUs in PCIe slots. Together they provide 152 RTX PRO 6000 GPUs with 14.25 TB of GPU memory and 17.8 PFLOP/s of FP32 compute.

    Per-Node Specifications¶

    Component Specification
    Server Dell PowerEdge XE7745 (4U, air-cooled + rear-door heat exchangers)
    CPUs 2x AMD EPYC 9555 (Turin, Zen 5) — 128 cores / 256 threads, 3.20 GHz
    GPUs 8x NVIDIA RTX PRO 6000 BSE (PCIe Gen5 x16, Blackwell GB202) — CC 12.0
    GPU Memory 96 GB GDDR7 per GPU — 768 GB per node
    GPU Memory Bandwidth 1,597 GB/s per GPU — 12.8 TB/s per node
    System RAM 24x 32 GB DDR5-6400 = 768 GB
    Local NVMe 1x 1.6 TB (mixed-use)
    FP32 per GPU 117 TFLOPS
    Tensor Cores 5th generation (FP4/FP8/FP6 micro-tensor scaling)
    Inter-GPU PCIe Gen5 x16 (~64 GB/s per GPU) — no NVLink
    Power ~7,900 W max per node
    Cooling Air-cooled with rear-door heat exchangers (RDHX)

    Network Interfaces¶

    Interface Speed Purpose
    2x ConnectX-7 NDR 400 Gb/s each Cluster IB fabric (VAST + MPI)
    2x 25 GbE 25 Gb/s each In-band management / provisioning
    1x 1 GbE (iDRAC) 1 Gb/s Out-of-band management

    RTX nodes have 2 InfiniBand connections to the shared cluster IB fabric — used for both storage access and inter-node MPI. Unlike the B200 nodes, there is no dedicated GPU data fabric.

    How RTX Differs from B200¶

    Property RTX PRO 6000 B200
    GPU Memory 96 GB GDDR7 180 GB HBM3e
    Memory Bandwidth 1,597 GB/s 8,000 GB/s
    FP32 per GPU 117 TFLOPS 75 TFLOPS
    Inter-GPU (intra-node) PCIe Gen5 (64 GB/s) NVLink 5 (1,800 GB/s)
    Inter-node fabric Shared cluster IB (100 GB/s) Dedicated 8-rail IB (400 GB/s)
    Compute Capability 12.0 10.0
    Cooling Air + RDHX Direct liquid
    Power per node ~7,900 W 12,200 W

    The RTX PRO 6000 actually has higher FP32 throughput than the B200. The B200's advantages are in memory bandwidth (5x), memory capacity (1.9x), and NVLink interconnect (28x faster than PCIe). The RTX excels at compute-bound workloads that fit within 96 GB per GPU.

    Different Compute Capabilities

    Despite both being Blackwell-generation, the B200 (CC 10.0, data-center family) and RTX PRO 6000 (CC 12.0, professional family) have distinct SM designs. Code compiled with nvcc should target the correct architecture for each tier.

    GDDR7 vs HBM3e¶

    Property GDDR7 (RTX) HBM3e (B200)
    Capacity per GPU 96 GB 180 GB
    Bandwidth per GPU 1,597 GB/s 8,000 GB/s
    Bus width Narrow, high-clocked (PAM4) 8192-bit, 3D-stacked
    Cost Moderate Very high

    GDDR7 uses PAM4 signaling to achieve ~1,600 GB/s — a 1.85x improvement over previous-generation GDDR6. For compute-bound workloads, GDDR7 is more than sufficient. For bandwidth-intensive workloads (large-batch inference, attention-heavy models), HBM3e has a clear advantage.
