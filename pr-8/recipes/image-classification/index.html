    DDP GPU Multi-Node PyTorch Recipes

    Distributed Image Classification¶

    Train a ResNet or Vision Transformer (ViT) across multiple GPU nodes using PyTorch Distributed Data Parallel (DDP) with torchrun.

    Goal¶

    Run distributed image classification training on 2 nodes (16 B200 GPUs total) using DDP, with ImageNet-style data loading and multi-node coordination via Slurm.

    Prerequisites¶

    • An AICR account with a valid Slurm account (--account=ACCOUNT_NAME)
    • A Conda environment with PyTorch and torchvision (see PyTorch on AICR)
    • Training data staged on /scratch/$USER or /work/INSTITUTION
    • Familiarity with Distributed Training concepts

    Steps¶

    1. Set Up the Environment¶

    $ conda create --prefix /scratch/$USER/conda-envs/vision python=3.12 -y
$ conda activate /scratch/$USER/conda-envs/vision
$ conda install pytorch torchvision torchaudio pytorch-cuda=12.4 -c pytorch -c nvidia -y
$ pip install timm

    The timm library provides pre-built ResNet and ViT model architectures.

    2. Stage the Dataset¶

    Copy your dataset to scratch for fast I/O during training:

    $ mkdir -p /scratch/$USER/datasets
$ cp -r /apps/datasets/imagenet/ /scratch/$USER/datasets/imagenet/

    The expected directory layout follows the ImageNet convention:

    imagenet/
  train/
    class_001/
      img_0001.JPEG
      img_0002.JPEG
    class_002/
      ...
  val/
    class_001/
      ...
    class_002/
      ...

    3. Write the Training Script¶

    Create /scratch/$USER/vision-demo/train_vision.py:

    """Distributed image classification training with DDP."""
import argparse
import os
import time

import torch
import torch.distributed as dist
import torch.nn as nn
from torch.nn.parallel import DistributedDataParallel as DDP
from torch.utils.data import DataLoader
from torch.utils.data.distributed import DistributedSampler
from torchvision import datasets, transforms
import timm


def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--data-dir", type=str, required=True)
    parser.add_argument("--model", type=str, default="resnet50",
                        help="Model name: resnet50, vit_base_patch16_224, etc.")
    parser.add_argument("--epochs", type=int, default=10)
    parser.add_argument("--batch-size", type=int, default=64,
                        help="Batch size per GPU")
    parser.add_argument("--lr", type=float, default=0.1)
    parser.add_argument("--output-dir", type=str, default=".")
    args = parser.parse_args()

    # --- Distributed setup ---
    dist.init_process_group(backend="nccl")
    local_rank = int(os.environ["LOCAL_RANK"])
    rank = int(os.environ["RANK"])
    world_size = int(os.environ["WORLD_SIZE"])
    torch.cuda.set_device(local_rank)
    device = torch.device("cuda", local_rank)

    if rank == 0:
        print(f"World size: {world_size}  |  Model: {args.model}")
        print(f"Effective batch size: {args.batch_size * world_size}")

    # --- Data ---
    train_transform = transforms.Compose([
        transforms.RandomResizedCrop(224),
        transforms.RandomHorizontalFlip(),
        transforms.ToTensor(),
        transforms.Normalize(mean=[0.485, 0.456, 0.406],
                             std=[0.229, 0.224, 0.225]),
    ])
    val_transform = transforms.Compose([
        transforms.Resize(256),
        transforms.CenterCrop(224),
        transforms.ToTensor(),
        transforms.Normalize(mean=[0.485, 0.456, 0.406],
                             std=[0.229, 0.224, 0.225]),
    ])

    train_dataset = datasets.ImageFolder(
        os.path.join(args.data_dir, "train"), transform=train_transform
    )
    val_dataset = datasets.ImageFolder(
        os.path.join(args.data_dir, "val"), transform=val_transform
    )

    train_sampler = DistributedSampler(train_dataset)
    train_loader = DataLoader(
        train_dataset,
        batch_size=args.batch_size,
        sampler=train_sampler,
        num_workers=8,
        pin_memory=True,
        persistent_workers=True,
    )
    val_loader = DataLoader(
        val_dataset,
        batch_size=args.batch_size,
        shuffle=False,
        num_workers=8,
        pin_memory=True,
    )

    # --- Model ---
    model = timm.create_model(args.model, pretrained=False,
                              num_classes=len(train_dataset.classes))
    model = model.to(device)
    model = DDP(model, device_ids=[local_rank])

    # Scale learning rate with world size (linear scaling rule)
    scaled_lr = args.lr * world_size
    optimizer = torch.optim.SGD(model.parameters(), lr=scaled_lr,
                                momentum=0.9, weight_decay=1e-4)
    scheduler = torch.optim.lr_scheduler.CosineAnnealingLR(
        optimizer, T_max=args.epochs
    )
    criterion = nn.CrossEntropyLoss()

    # --- Training loop ---
    for epoch in range(1, args.epochs + 1):
        model.train()
        train_sampler.set_epoch(epoch)
        epoch_loss = 0.0
        correct = 0
        total = 0
        start = time.time()

        for images, labels in train_loader:
            images, labels = images.to(device), labels.to(device)

            optimizer.zero_grad()
            with torch.amp.autocast(device_type="cuda", dtype=torch.bfloat16):
                outputs = model(images)
                loss = criterion(outputs, labels)
            loss.backward()
            optimizer.step()

            epoch_loss += loss.item() * images.size(0)
            _, predicted = outputs.max(1)
            correct += predicted.eq(labels).sum().item()
            total += labels.size(0)

        scheduler.step()
        elapsed = time.time() - start

        if rank == 0:
            train_acc = 100.0 * correct / total
            print(f"Epoch {epoch:>3d}/{args.epochs}  "
                  f"loss={epoch_loss / total:.4f}  "
                  f"acc={train_acc:.1f}%  "
                  f"lr={scheduler.get_last_lr()[0]:.6f}  "
                  f"[{elapsed:.1f}s]")

    # --- Save final model ---
    if rank == 0:
        os.makedirs(args.output_dir, exist_ok=True)
        ckpt_path = os.path.join(args.output_dir, "model_final.pt")
        torch.save(model.module.state_dict(), ckpt_path)
        print(f"Model saved to {ckpt_path}")

    dist.destroy_process_group()


if __name__ == "__main__":
    main()

    4. Multi-Node Job Script (2 Nodes, 16 GPUs)¶

    Create distributed_vision.sh:

    #!/bin/bash
#SBATCH --job-name=dist-vision
#SBATCH --partition=b200-batch
#SBATCH --nodes=2
#SBATCH --ntasks-per-node=1
#SBATCH --gpus-per-node=8
#SBATCH --cpus-per-task=64
#SBATCH --mem=0
#SBATCH --time=12:00:00
#SBATCH --account=ACCOUNT_NAME
#SBATCH --output=%x-%j.out

module purge

eval "$(conda shell.bash hook)"
conda activate /scratch/$USER/conda-envs/vision

export MASTER_ADDR=$(scontrol show hostnames $SLURM_JOB_NODELIST | head -n 1)
export MASTER_PORT=29500

srun torchrun \
    --nnodes=$SLURM_NNODES \
    --nproc_per_node=8 \
    --rdzv_id=$SLURM_JOB_ID \
    --rdzv_backend=c10d \
    --rdzv_endpoint=$MASTER_ADDR:$MASTER_PORT \
    /scratch/$USER/vision-demo/train_vision.py \
        --data-dir /scratch/$USER/datasets/imagenet \
        --model resnet50 \
        --epochs 10 \
        --batch-size 64 \
        --lr 0.1 \
        --output-dir /scratch/$USER/vision-demo/results

    Note

    With 2 nodes and 8 GPUs per node, torchrun launches 16 processes total. The effective batch size is 64 * 16 = 1024. The learning rate is scaled by world_size in the training script (linear scaling rule).

    5. Submit and Monitor¶

    $ sbatch distributed_vision.sh
Submitted batch job 12345

    Monitor the job:

    $ squeue -u $USER
$ tail -f dist-vision-12345.out

    To switch to a ViT model, change --model resnet50 to --model vit_base_patch16_224.

    Job Script¶

    The complete 2-node job script for reference:

    #!/bin/bash
#SBATCH --job-name=dist-vision
#SBATCH --partition=b200-batch
#SBATCH --nodes=2
#SBATCH --ntasks-per-node=1
#SBATCH --gpus-per-node=8
#SBATCH --cpus-per-task=64
#SBATCH --mem=0
#SBATCH --time=12:00:00
#SBATCH --account=ACCOUNT_NAME
#SBATCH --output=%x-%j.out

module purge

eval "$(conda shell.bash hook)"
conda activate /scratch/$USER/conda-envs/vision

export MASTER_ADDR=$(scontrol show hostnames $SLURM_JOB_NODELIST | head -n 1)
export MASTER_PORT=29500

srun torchrun \
    --nnodes=$SLURM_NNODES \
    --nproc_per_node=8 \
    --rdzv_id=$SLURM_JOB_ID \
    --rdzv_backend=c10d \
    --rdzv_endpoint=$MASTER_ADDR:$MASTER_PORT \
    /scratch/$USER/vision-demo/train_vision.py \
        --data-dir /scratch/$USER/datasets/imagenet \
        --model resnet50 \
        --epochs 10 \
        --batch-size 64 \
        --lr 0.1 \
        --output-dir /scratch/$USER/vision-demo/results

    Troubleshooting¶

    Problem Cause Fix
    NCCL error: remote process exited One node failed to join the process group Verify both nodes are allocated (squeue -j JOBID); check that MASTER_ADDR resolves correctly
    Training hangs at startup Port conflict or firewall issue Try a different MASTER_PORT (e.g., 29501)
    RuntimeError: DataLoader worker exited unexpectedly Too many workers for available CPUs Reduce num_workers or increase --cpus-per-task
    Low GPU utilization Data loading bottleneck Increase num_workers, use persistent_workers=True, or stage data to scratch for faster I/O
    Accuracy does not improve with more GPUs Learning rate not scaled Apply the linear scaling rule: multiply base LR by world_size
    FileNotFoundError for dataset Data not staged to scratch Copy dataset to /scratch/$USER/datasets/ before running
